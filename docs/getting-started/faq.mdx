# F.A.Q.s

### Who should use WasmKit?

Anyone who is writing smart contracts in CosmWasm on any Cosmos based blockchain.

### Why use WasmKit?

WasmKit saves dev time and less error-prone while working with CosmWasm contracts.

### Does WasmKit support the chain I'm working with?

WasmKit supports multiple chains by default and it can used with any chain of choice, just need to add the chain's configuration to `wasmkit.conig.js` and `wasmkit.chain.js`.

### WasmKit is missing a feature I require, how to add?

You can open an issue on our [Github repository](https://github.com/kubiklabs/wasmkit) or make a pull request.
